### Installation
- Clone repository
```
 git clone https://github.com/bk52/socialityio.git
 ```
- Enter folder
```
cd socialityio
```
- Build docker image
```
docker build . -t socialityio
```
- Run docker image
```
docker run -p 9200:80 socialityio
```
- Open [http://localhost:9200/](http://localhost:9200/) in browser
